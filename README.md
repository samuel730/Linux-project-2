## 1. Introduction

This lab practical was carried out to understand and apply Linux file and directory permissions using command-line tools. The objective of this project was to practice modifying file permissions, managing ownership, creating users and groups, applying special permissions (Setuid, Setgid, Sticky Bit), and controlling default permissions using umask.

The lab was performed inside a virtualized Linux environment using Vagrant and Ubuntu Linux.
